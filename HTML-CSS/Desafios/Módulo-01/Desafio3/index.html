Desafio do Mapa Mundi

Aqui, estou treinando o uso de imagens, parágrafos e quebras de linhas.

Montei o Mapa Mundi com 9 imagens diferentes.



Posso fazer isso de dois jeitos:
1: Dividindo em 3 parágrafos diferentes, com 3 imagens cada, o que gera uma separação de linhas maior.
2: Como eu preferi fazer, usando a tag <br>a cada 3 imagens, para gerar uma quebra de linha. Dessa forma as imagens tem uma separação por linha mais próxima.
