Parágrafos e quebras de linha

Você pode descrever um parágrafo de qualquer jeito. Basta colocar tudo entre o par de tags <p> e </p>.

Se precisar quebrar o texto em algum lugar específico
como esse, você pode usar a tag <br>.

Vamos adicionar alguns símbolos especiais:
1:   - Espaço em Branco
2: ® - Marca Registrada
3: © - Copyright
4: ™ -Trade Mark
5: €, £, ¥, ¢ - Euro, Libra(pound), Yen, Cent
6: ∅ - Vazio
7: ∑ - Somatório
8: Δ - Delta
9: ←, ↑, →, ↓ - Setas de direção (left, up, right, down)

Vamos adicionar também, alguns emojis:
😀
🤪
🐒
🏆
Para utilizarmos algum emoji, devemos colocar o prefixo (&#x), para indicar que estamos falando de Hexadecimal. Depois colocamos o código do emoji.
